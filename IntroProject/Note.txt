4) C++과 블루프린트
 - 블루프린트 장점 
	1. 빠른 수정 후 엔진에 적용 가능
	2. 초보자가 사용하기 좋다(전공 지식없이 배우기 쉬움)
	3. 내장 툴이 있어서 기능 검색하기 좋다
	4. 언리얼 전용이라 충돌없이 잘 맞는다
	5. 디자이너와 아티스트와 협력하기 좋다(설명 쉽고 함께 사용 가능)
 - C++ 장점
	1. 훨씬 간결하다
	2. 업계 표준 언어이기 때문에 쓰는 사람이 많아 툴과 정보를 찾기 쉽다
	3. 속도가 매우 빠르다
	4. 언리얼의 모든 영역과 기능에 접근 할 수 있다
	5. 대형 게임에 적합하다(블루프린트는 실수하기 쉬움)

- 언리얼 라이브러리는 C++로 제작됐으나 블루프린트로도 접근 가능
- 직접 C++로 함수를 작성해도 블루프린트로 사용가능
- 즉, 블루프린트와 C++를 함께 사용하면 두 장점을 모두 흡수 가능
